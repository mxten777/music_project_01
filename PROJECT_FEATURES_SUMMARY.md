# AI 작사·작곡 도우미 프로젝트 기능 요약

## 전체 개요
- 감성적인 시(텍스트)를 입력하면 AI가 가사, 코드 진행, 샘플 멜로디를 자동 생성해주는 웹앱입니다.
- Vite + React + TailwindCSS 기반, (향후 Firebase/AI API 연동 확장 예정)

## 전체 주요 기능
1. **시(텍스트) 입력 및 가사 변환**
   - 사용자가 시를 입력하거나 예시를 불러올 수 있음
   - [변환] 버튼 클릭 시 AI가 가사로 변환
2. **코드 진행 자동 추천**
   - AI가 입력된 시/가사에 어울리는 코드 진행(예: Am–F–C–G 등) 자동 추천
3. **샘플 멜로디 오디오 프리뷰**
   - AI가 생성한 멜로디를 오디오로 미리 듣기/다운로드 가능
4. **결과 복사/다운로드**
   - 가사/코드/멜로디 결과를 복사하거나 오디오 파일로 저장 가능
5. **피드백/의견 제출**
   - 사용자 피드백 입력 및 리스트 확인
6. **다크모드 지원, 반응형 UI, 접근성, Storybook 문서화**

## 메뉴별 기능
- **메인(Main)**
  - 시 입력 폼, 예시 불러오기, 변환 버튼
  - 변환 결과(가사, 코드, 멜로디) 표시
  - 결과 복사, 오디오 다운로드
- **피드백(Feedback)**
  - 피드백 입력 폼(시/의견)
  - 제출된 피드백 리스트 확인
- **GitHub**
  - 프로젝트 깃허브 저장소로 이동(외부 링크)

## 주요 페이지/컴포넌트
- `MainPage.tsx`: 전체 메인 플로우(시 입력, 결과, 피드백)
- `PoemInput.tsx`: 시 입력 폼
- `LyricsResult.tsx`: 변환 결과(가사/코드/멜로디)
- `AudioPreview.tsx`: 멜로디 오디오 프리뷰
- `FeedbackForm.tsx`, `FeedbackList.tsx`: 피드백 입력/리스트
- `Header.tsx`: 상단 네비게이션(메뉴, 다크모드, 모바일 드로어)
- `Footer.tsx`: 하단 정보
- `UsageGuideModal.tsx`: 사용법 안내(우측 하단)

## 실행 방법
```bash
npm install
npm run dev
```
